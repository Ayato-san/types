### Installation

- with npm
  ```sh
  npm install -D @ayato-san/types
  ```

- with pnpm
  ```sh
  pnpm install -D @ayato-san/types
  ```

- with yarn
  ```sh
  yarn add -D @ayato-san/types
  ```
